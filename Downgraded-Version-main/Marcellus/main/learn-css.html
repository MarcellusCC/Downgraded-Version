Selectors & Specificity

Basic selectors and how specificity wins conflicts.
/* Type, class, ID */
p { color: #555; }         /* specificity 0-0-0-1 */
.note { color: #333; }     /* 0-0-1-0 */
#urgent { color: #000; }   /* 0-1-0-0 */

/* Attribute & pseudo-class */
a[target="_blank"] { text-decoration: underline; }
a:hover { color: #ff8c00; }

/* More specific rule overrides less specific */
#urgent.note { color: red; }

The Cascade: Source Order & Importance

Later rules of the same specificity override earlier ones; !important trumps normal rules.
.msg { color: #666; }
.msg { color: #222; } /* later wins */

/* Use !important sparingly */
.msg { color: #999 !important; }
