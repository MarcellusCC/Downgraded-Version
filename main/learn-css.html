Selectors, colors, text, the box model, Flexbox & Grid, responsiveness.

1 Selectors

Target elements using tag, class .class, id #id, or attribute selectors.

h1 { color: orange; } .title { font-weight: 700; } #banner { background: #111; }

This text turns orange via h1 style example.

2 Colors & Typography

Use hex, rgb(a), hsl. Control font-size, weight, line-height, letter-spacing.

p { color: hsl(30, 100%, 50%); font-size: 1.1rem; line-height: 1.7; }

Nice warm orange paragraph.

3 Box Model

Elements have content, padding, border, and margin. Use box-sizing:border-box to simplify sizing.

.box { padding: 12px; border: 2px solid #ff8c00; margin: 10px; box-sizing: border-box; }
Box example

4 Flexbox

Create 1-D layouts. Align, justify, and order items easily.

.row { display: flex; gap: 10px; justify-content: space-between; }
